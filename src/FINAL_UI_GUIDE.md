## 🎯 **주요 개선 사항**

### **1. 탭 통합**

```
Before (4개 탭):
- 기본 정보
- Step 관리
- Visibility (별도)
- 이미지 업로드 (별도)

After (2개 탭):
- 기본 정보
- Step 관리 (모든 기능 통합)
  ├─ Step 제목/설명
  ├─ 표시/숨김 (Visibility)
  ├─ 이미지 업로드 (라디오 버튼)
  └─ 이미지 미리보기
```

**효과:**
- 탭 개수: 4개 → 2개 (50% 감소!)
- 탭 전환: 거의 0회
- 모든 작업이 한 곳에서 완료

---

### **2. 라디오 버튼 이미지 입력**

#### **기본 선택: 이미지 파일 업로드**

```typescript
// 기본값
imageInputMethod: "upload" (또는 undefined)

// UI
⦿ 이미지 파일 업로드 [권장]
○ URL 주소 입력
```

**특징:**
- 기본값이 "파일 업로드"
- "권장" 배지 표시 (녹색)
- 드래그 앤 드롭 UI
- 권장 사이즈 안내: PNG, 400×800px

#### **대안: URL 주소 입력**

```typescript
// URL 모드
imageInputMethod: "url"

// UI
○ 이미지 파일 업로드 [권장]
⦿ URL 주소 입력
```

**특징:**
- ��부 이미지 URL 직접 입력
- Unsplash, Figma asset 등 사용 가능
- 간단한 Input 필드

### **3. 권장 사이즈 안내**

```
💡 권장 사이즈: PNG, 400×800px
```

**위치:**
- 파일 업로드 모드 하단

- 400×800px: 모바일 최적화 세로형 비율
- 매뉴얼 페이지에 적합한 사이즈

## 🎯 **사용 방법**

### **시나리오 1: 새 Step 추가 + 파일 업로드**

```
1. 📋 Step 관리 탭
2. "Step 추가" 버튼
3. 제목/설명 입력
4. 이미지 섹션:
   ⦿ 이미지 파일 업로드 [권장] (기본 선택됨)
5. 이미지 드래그 앤 드롭 또는 클릭
6. PNG, 400×800px 권장 확인
7. 미리보기 확인
8. ☑ 매뉴얼에 표시
9. ☑ 이미지 표시
10. 💾 저장

→ 탭 전환 0회! ✅
```

### **시나리오 2: URL로 이미지 추가**

```
1. 📋 Step 관리 탭
2. "Step 추가" 버튼

4. 이미지 섹션:
   ○ URL 주소 입력 (클릭)
5. URL 입력: https://images.unsplash.com/...
6. 미리보기 자동 표시
7. ☑ 매뉴얼에 표시
8. ☑ 이미지 표시
9. 💾 저장

→ 간단하고 빠름! ✅
```

### **시나리오 3: 파일 업로드 → URL로 변경**

```
1. 📋 Step 관리 탭
2. 기존 Step 카드
3. 이미지 섹션:
   ⦿ 이미지 파일 업로드 [권장]
   ↓ 클릭
   ○ URL 주소 입력
4. URL 입력
5. 이전 업로드 이미지는 그대로 유지 (덮어쓰기 안 됨)
6. 💾 저장

→ 자유롭게 전환 가능! ✅
```

## 🔍 **기술적 세부사항**

### **imageInputMethod 상태:**

```typescript
interface StepData {
  imageInputMethod?: "upload" | "url";
}

// 기본값 (undefined 또는 "upload")
step.imageInputMethod !== "url" → 파일 업로드 모드

// URL 모드
step.imageInputMethod === "url" → URL 입력 모드
```

### **라디오 버튼 로직:**

```typescript
// 파일 업로드 선택
<input
  type="radio"
  name={`image-input-${index}`}
  checked={step.imageInputMethod !== "url"}
  onChange={() => {
    newSteps[index].imageInputMethod = "upload";
  }}
/>

// URL 입력 선택
<input
  type="radio"
  name={`image-input-${index}`}
  checked={step.imageInputMethod === "url"}
  onChange={() => {
    newSteps[index].imageInputMethod = "url";
  }}
/>
```

### **조건부 렌더링:**

```typescript
// 파일 업로드 모드
{step.imageInputMethod !== "url" && (
  <ImageUploader ... />
)}

// URL 입력 모드
{step.imageInputMethod === "url" && (
  <Input ... />
)}
```

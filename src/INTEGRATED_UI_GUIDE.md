### **3. 새로운 기능**

#### **✅ 실시간 Visibility 표시:**

```typescript
<Card className={`${!step.visible ? 'opacity-60' : ''}`}>
  {/* Step이 숨겨져 있으면 카드가 흐릿하게 표시 */}
</Card>
```

**효과:**
- 매뉴얼에 표시되지 않는 Step은 흐릿하게 표시
- 시각적으로 즉시 확인 가능

#### **✅ 매뉴얼에 표시 체크박스:**

```tsx
<label className="flex items-center gap-2 cursor-pointer">
  <input
    type="checkbox"
    checked={step.visible}
    onChange={(e) => {
      const newSteps = [...pageData.steps];
      newSteps[index].visible = e.target.checked;
      setPageData({ ...pageData, steps: newSteps });
    }}
  />
  <span className="text-muted-foreground">매뉴얼에 표시</span>
</label>
```

**기능:**
- 체크: Step이 매뉴얼에 표시됨
- 해제: Step이 매뉴얼에서 숨겨짐

#### **✅ 이미지 표시 체크박스:**

```tsx
{step.image && (
  <label className="flex items-center gap-2 cursor-pointer">
    <input
      type="checkbox"
      checked={step.imageVisible}
      onChange={(e) => {
        const newSteps = [...pageData.steps];
        newSteps[index].imageVisible = e.target.checked;
        setPageData({ ...pageData, steps: newSteps });
      }}
      disabled={!step.image}
    />
    <span className="text-muted-foreground">이미지 표시</span>
  </label>
)}
```

**기능:**
- 이미지가 있을 때만 표시
- 체크: 이미지가 매뉴얼에 표시됨
- 해제: 이미지가 매뉴얼에서 숨겨짐

#### **✅ 이미지 미리보기:**

```tsx
{step.image && (
  <div className="mt-3">
    <img
      src={step.image}
      alt={`Step ${step.number} preview`}
      className="w-full max-w-md rounded-lg border border-border"
    />
  </div>
)}
```

**기능:**
- 이미지 URL 입력 시 즉시 미리보기 표시
- 업로드된 이미지 확인 가능

## 🎨 **UI/UX 개선 포인트**

### **1. 직관적인 레이아웃**

```
이전:
- Step 제목/설명 편집 → Step 관리 탭
- Step 표시/숨김 → Visibility 탭 (별도 이동 필요)
- 이미지 확인 → 이미지 업로드 탭 (별도 이동 필요)

현재:
- 모든 Step 관련 작업 → Step 관리 탭 (한 곳에서 완료)
  ├─ 제목/설명 편집
  ├─ 표시/숨김 설정
  ├─ 이미지 업로드
  └─ 이미지 미리보기
```

### **2. 시각적 피드백**

```
✅ 숨겨진 Step → 카드 흐릿하게 (opacity-60)
✅ 이미지 있음 → "이미지 표시" 체크박스 활성화
✅ 이미지 없음 → "이미지 표시" 체크박스 비활성화
✅ 이미지 URL 입력 → 즉시 미리보기 표시
```

### **3. 효율적인 워크플로우**

```
Step 추가/편집 워크플로우:

1. "Step 추가" 버튼 클릭
2. 제목 입력 (한국어/영어)
3. 설명 입력 (한국어/영어)
4. 이미지 업로드 또는 URL 입력
5. 이미지 미리보기 확인
6. "매뉴얼에 표시" 체크 (필요 시)
7. "이미지 표시" 체크 (필요 시)
8. 저장 버튼

→ 모든 작업이 한 화면에서 완료! ✅
```

|--------|------|-----------|
| 📝 기본 정보 | 제목, 소개 | ⭐⭐⭐ |
| 📋 Step 관리 | Step 편집 | ⭐⭐⭐⭐⭐ |

| 탭 이름 | 기능 | 사용 빈도 |

| 📝 기본 정보 | 제목, 소개, 헤더 이미지 | ⭐⭐⭐ |

**개선점:**
- Step 관련 모든 작업이 하나의 탭에 통합
- 탭 전환 최소화
- 사용 빈도가 높은 기능이 집중됨

## 🔍 **기술적 세부사항**

### **1. 조건부 opacity**

```typescript
<Card className={`${!step.visible ? 'opacity-60' : ''}`}>
```

**효과:**
- `step.visible === false` → 카드가 60% 투명도
- 시각적으로 "비활성화" 느낌

### **2. 조건부 체크박스**

```typescript
{step.image && (
  <label>
    <input type="checkbox" />
    <span>이미지 표시</span>
  </label>
)}
```

**효과:**
- 이미지가 없으면 체크박스 자체가 숨겨짐
- 불필요한 UI 요소 제거

### **3. 조건부 미리보기**

```typescript
{step.image && (
  <div className="mt-3">
    <img src={step.image} />
  </div>
)}
```

**효과:**
- 이미지 URL이 있을 때만 미리보기 표시
- 실시간 반영


### **클릭 횟수:**

```

  1. Visibility 탭 클릭
  2. 체크박스 클릭
  3. 저장 버튼 클릭

현재: 2회 클릭
  1. 체크박스 클릭 (Step 카드에서 바로)
  2. 저장 버튼 클릭

→ 33% 감소! ✅
```
